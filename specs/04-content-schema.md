# Content Schema for Joe the Painter: A Rebirth

## Content Strategy Overview

The content strategy for Joe the Painter is designed to establish him as the preeminent, trustworthy, and experienced craftsman for painting and handyman services in San Diego County. Moving beyond a generic service provider, we will tell the story of Joe himself – his two decades of licensed expertise, his personal commitment to quality, and the peace of mind he delivers. This rebirth focuses on a personal brand built on trust, legacy, and meticulous craftsmanship.

**Content Pillars:**

1.  **The Craftsman's Guarantee:** Emphasizing Joe's personal involvement, attention to detail, and a quarter-century legacy of quality. Every piece of content reinforces Joe's personal accountability and dedication.
2.  **Integrated Solutions:** Showcasing the unique value of a licensed painter who also expertly handles essential handyman repairs, offering a seamless, comprehensive service that solves multiple homeowner problems.
3.  **Hyper-Local Expertise:** Reinforcing Joe's deep roots and specific service within Central and North San Diego County, fostering a sense of community and local reliability.
4.  **Transparency & Trust:** Providing clear information, verifying credentials, showcasing authentic customer experiences, and demystifying the project process.

**How Content Supports Business Goals:**

*   **Lead Generation:** Clear calls-to-action (Free Consultation), detailed service descriptions, and location-specific content will drive qualified leads directly to Joe.
*   **Brand Building:** A consistent, personal narrative across all content will differentiate Joe as a trusted master craftsman, not just another contractor.
*   **SEO:** Structured service pages, rich project case studies, and comprehensive FAQ content will target high-intent keywords, answer user queries, and establish Joe's authority in local search results.
*   **Conversion:** Authentic testimonials, compelling before-and-after galleries, and transparent process explanations will build confidence and encourage prospective clients to take the next step.

---

## Common Content Types

Throughout the schemas, `RichText` is used as a type. This represents content that can include formatting (bold, italics), lists, links, and potentially embedded media. Implementations may use Markdown, a rich-text editor, or similar.

## Collection: Service

### Purpose
This collection defines all painting and handyman services offered by Joe the Painter. Each entry will serve as a dedicated landing page, detailing the scope, benefits, process, and visual examples of the service. This is critical for clear communication, SEO, and guiding potential clients.

### Schema

```typescript
interface Service {
  id: string; // Unique identifier (e.g., "interior-painting")
  title: string; // e.g., "Interior Painting Services"
  slug: string; // e.g., "interior-painting"
  shortDescription: string; // Concise summary for cards/listings
  longDescription: RichText; // Detailed description of the service, including what's involved and client benefits
  category: 'Painting' | 'Handyman'; // Primary service category
  serviceTypes: string[]; // Specific types within category, e.g., ["Residential", "Commercial", "Cabinet Refinishing"]
  benefits: string[]; // Key advantages of choosing Joe for this specific service
  processSteps: {
    title: string;
    description: string;
    image?: { src: string; alt: string; }; // Optional image for each step
  }[]; // Step-by-step process for the service, demystifying the work
  featuredImage: { // Main image for the service page hero and cards
    src: string;
    alt: string;
    caption?: string;
  };
  galleryImages?: { // Additional visual examples of the service
    src: string;
    alt: string;
    caption?: string;
  }[];
  faqs?: { // Service-specific FAQs, embedded for context
    question: string;
    answer: RichText;
  }[];
  relatedProjects?: string[]; // Array of Project slugs related to this service
  ctaText: string; // Call-to-action text specific to this service (e.g., "Get a Free Interior Painting Estimate")
  seo: {
    title: string;
    description: string;
    keywords?: string[]; // Specific keywords for this service page
  };
  published: boolean; // Controls visibility on the live site
}
```

### Field Definitions

| Field            | Type       | Required | Validation                                | Description                                                                                                                                                                                                                                |
| :--------------- | :--------- | :------- | :---------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `id`             | string     | Yes      | kebab-case, unique                      | A unique, stable identifier for the service (e.g., `interior-painting`).                                                                                                                                                                   |
| `title`          | string     | Yes      | max 100 chars                             | The primary display title for the service page and navigation.                                                                                                                                                                             |
| `slug`           | string     | Yes      | kebab-case, unique                      | The URL-friendly slug for the service page (e.g., `/services/interior-painting`).                                                                                                                                                          |
| `shortDescription` | string     | Yes      | max 200 chars                             | A brief, compelling summary of the service, used in service listings or card previews.                                                                                                                                                      |
| `longDescription` | RichText   | Yes      | min 200 chars                             | The comprehensive description of the service, detailing its scope, what clients can expect, and Joe's approach to quality and craftsmanship.                                                                                                 |
| `category`       | enum       | Yes      | `Painting` \| `Handyman`                  | The primary category this service falls under, used for filtering and navigation.                                                                                                                                                          |
| `serviceTypes`   | string[]   | Yes      | min 1 entry, max 5, predefined tags     | Specific sub-types or applications of the service (e.g., `Residential`, `Cabinet Refinishing`).                                                                                                                                            |
| `benefits`       | string[]   | Yes      | min 3 entries, max 8                      | A list of key benefits or advantages a client receives when choosing Joe for this service (e.g., "Flawless Finish," "Durable, Long-Lasting Results").                                                                                      |
| `processSteps`   | array      | No       | min 2 steps if present                  | An ordered list of steps outlining Joe's process for this service, enhancing transparency and trust. Each step can have a `title`, `description`, and an optional `image`.                                                                 |
| `featuredImage`  | object     | Yes      | `src`, `alt` required; `caption` optional | The main visual asset representing the service, used prominently on the service page and in listings.                                                                                                                                      |
| `galleryImages`  | array      | No       | min 1 image if present, max 10            | A collection of additional high-quality images showcasing completed work or aspects of the service.                                                                                                                                        |
| `faqs`           | array      | No       | min 1 FAQ if present, max 5               | A set of frequently asked questions directly relevant to this specific service, with detailed answers.                                                                                                                                     |
| `relatedProjects` | string[]   | No       | array of `Project` slugs                | A list of slugs for related `Project` entries, linking directly to Joe's portfolio for visual proof.                                                                                                                                       |
| `ctaText`        | string     | Yes      | max 80 chars                              | The clear, concise call-to-action text displayed prominently on the service page, guiding users to the next step.                                                                                                                          |
| `seo`            | object     | Yes      | `title` (max 70), `description` (max 160) | Structured metadata for search engine optimization, including a compelling `title` and `description` to improve click-through rates. Optional `keywords` for internal reference.                                                              |
| `published`      | boolean    | Yes      | default: `false`                          | A flag to control whether the service page is live and visible on the website.                                                                                                                                                             |

### Relationships
*   **Related to:** `Project` (through `relatedProjects` field).
*   **Referenced by:** Homepage (featured services), Services landing page, Project detail pages, Footer navigation.

### Display Contexts
*   **List view (Services landing page):** Displays `featuredImage`, `title`, `shortDescription`, and `ctaText` in a card or grid format.
*   **Detail view (Service page):** Full page display of all fields, including `longDescription`, `processSteps`, `galleryImages`, and embedded `faqs`.
*   **Card preview (Homepage, related sections):** A concise card featuring `featuredImage`, `title`, and `shortDescription`.

## Collection: Project

### Purpose
This collection serves as Joe's portfolio, showcasing completed work in a case-study format. Each project entry tells a story, highlighting the client's challenge, Joe's solution, and the achieved results, often with compelling before-and-after visuals. This builds trust, demonstrates expertise, and provides tangible proof of craftsmanship.

### Schema

```typescript
interface Project {
  id: string; // Unique identifier (e.g., "encinitas-coastal-home-exterior")
  title: string; // e.g., "Encinitas Coastal Home Exterior Refresh"
  slug: string; // e.g., "encinitas-coastal-home-exterior"
  projectDate: Date; // Completion date of the project (YYYY-MM-DD)
  clientName?: string; // Optional: "Homeowner in Encinitas" (for privacy, not full name)
  location: string; // e.g., "Encinitas, CA"
  shortDescription: string; // Concise summary for gallery cards
  challenge: RichText; // The problem or need the client presented
  solution: RichText; // How Joe addressed the challenge, detailing process and expertise
  result: RichText; // The positive outcome for the client and their property
  servicesPerformed: string[]; // Array of Service IDs or slugs related to this project
  beforeImages?: { // Optional: Images showing the state before Joe's work
    src: string;
    alt: string;
  }[];
  afterImages: { // Images showing the completed work
    src: string;
    alt: string;
  }[];
  featuredImage: { // Main image for project card/hero
    src: string;
    alt: string;
    caption?: string;
  };
  testimonial?: string; // Optional: Direct quote from the client for this project
  relatedServices?: string[]; // Array of Service slugs relevant to this project
  seo: {
    title: string;
    description: string;
  };
  published: boolean; // Controls visibility on the live site
}
```
